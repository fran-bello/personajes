# Sonidos del Juego

Esta carpeta contiene los archivos de sonido para el juego Personajes.

## Sonidos Requeridos

Puedes agregar los siguientes archivos de sonido para mejorar la experiencia del juego:

- `hit.mp3` - Sonido cuando se acierta un personaje
- `fail.mp3` - Sonido cuando se falla un personaje
- `timeup.mp3` - Sonido cuando se acaba el tiempo
- `tick.mp3` - Sonido de tick del timer (últimos segundos)
- `round-start.mp3` - Sonido al iniciar una nueva ronda
- `game-start.mp3` - Sonido al iniciar una partida
- `game-end.mp3` - Sonido al terminar una partida
- `card-flip.mp3` - Sonido al voltear una tarjeta
- `pause.mp3` - Sonido al pausar el juego
- `resume.mp3` - Sonido al reanudar el juego

## Formatos Soportados

El juego soporta los siguientes formatos de audio:
- MP3 (recomendado para compatibilidad)
- WAV
- OGG

## Notas

- Si no agregas los archivos de sonido, el juego usará sonidos generados por defecto usando Web Audio API
- Los sonidos se pre-cargan automáticamente cuando se inicia el juego
- Puedes deshabilitar los sonidos desde las preferencias del usuario (si se implementa)
